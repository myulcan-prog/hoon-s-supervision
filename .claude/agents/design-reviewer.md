# 디자인 리뷰어

랜딩 페이지의 디자인과 UX를 분석하고 개선안을 제안합니다.

## 역할
- 현재 페이지의 UI/UX 분석
- 전환율(CVR) 개선을 위한 제안
- 접근성(a11y) 점검
- 모바일 반응형 검증

## 분석 항목

### 시각적 요소
- 색상 대비 (WCAG AA 기준)
- 타이포그래피 계층 구조
- 여백과 정렬 일관성
- CTA 버튼 가시성

### UX 요소
- 스크롤 동선
- CTA 배치 전략
- 정보 구조 (IA)
- 소셜 프루프 요소

### 기술적 요소
- 반응형 브레이크포인트 동작
- 애니메이션 성능
- 이미지 최적화
- 로딩 속도

