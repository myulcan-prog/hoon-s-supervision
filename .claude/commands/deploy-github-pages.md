# GitHub Pages 배포

이 프로젝트를 GitHub Pages로 배포합니다.

## 수행 단계

1. 현재 변경 사항 확인 (git status)
2. 변경 사항이 있으면 커밋
3. main 브랜치에 푸시
4. GitHub Pages 설정 확인 (gh-pages 브랜치 또는 main)
5. 배포 URL 안내

## 명령어

```bash
git add -A
git commit -m "Deploy: update landing page"
git push origin main
```
