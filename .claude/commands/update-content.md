# 콘텐츠 업데이트

랜딩 페이지의 텍스트, 가격, 커리큘럼 등 콘텐츠를 업데이트합니다.

## 수행 단계

1. 사용자에게 변경할 섹션과 내용 확인
2. index.html에서 해당 섹션 찾기
3. 콘텐츠 수정
4. 변경 전후 비교 안내

## 변경 가능한 섹션
- Hero: 메인 타이틀, 서브 타이틀, 통계
- Problem: 문제 카드 3개
- Solution: 솔루션 3개
- Curriculum: 12회기 커리큘럼
- Pricing: 가격 정보
- Supervisor: 수퍼바이저 정보
- FAQ: 자주 묻는 질문
- CTA: 신청 링크

## 주의사항
- 기존 디자인 톤앤매너 유지
- HTML 구조 변경 최소화
- 반응형 레이아웃에 영향 없도록 주의
