# 훈스아카데미 — ABA 온라인 수퍼비전 랜딩 페이지

## 프로젝트 개요
- ABA(응용행동분석) 온라인 수퍼비전 프로그램 홍보용 랜딩 페이지
- 단일 페이지(SPA) 정적 HTML 프로젝트
- 한국어 기반 서비스

## 기술 스택
- HTML5 + CSS3 + Vanilla JavaScript
- Google Fonts (Noto Sans KR, Playfair Display)
- 빌드 도구 없음 (정적 파일)

## 프로젝트 구조
```
/
├── index.html          # 메인 랜딩 페이지 (HTML + CSS + JS 올인원)
├── CLAUDE.md           # Claude Code 설정
└── .claude/            # Claude Code 설정 폴더
    ├── settings.local.json
    ├── commands/       # 슬래시 커맨드
    └── agents/         # 에이전트
```

## 페이지 섹션 구조
1. **Nav** — 고정 네비게이션 (스크롤 시 그림자)
2. **Hero** — 메인 히어로 (배지, 타이틀, CTA, 통계)
3. **Problem** — 문제 제기 (3개 카드)
4. **Solution** — 해결 방법 (3가지 솔루션)
5. **Target** — 추천 대상 (4개 항목)
6. **Flow** — 수퍼비전 1회기 흐름 (4단계)
7. **Curriculum** — 6개월 커리큘럼 (12회기)
8. **Outcome** — 6개월 후 성과 (4개 카드)
9. **Pricing** — 가격 (1:1 / 그룹)
10. **Supervisor** — 수퍼바이저 소개
11. **FAQ** — 자주 묻는 질문 (5개)
12. **CTA** — 최종 신청 유도
13. **Footer** — 푸터

## 디자인 시스템
- **주 색상**: `--accent: #2D5A27` (녹색)
- **보조 색상**: `--accent-warm: #D4A853` (골드)
- **배경**: `--bg: #FAFAF7` (아이보리)
- **반응형**: 900px / 640px 브레이크포인트
- **애니메이션**: scroll reveal (IntersectionObserver)

## 코딩 컨벤션
- 한국어 텍스트 사용 (서비스 대상: 한국인)
- CSS 변수(Custom Properties) 사용
- BEM 네이밍 대신 시맨틱 클래스명 사용
- 모바일 우선이 아닌 데스크톱 우선 반응형

## 사업 컨텍스트
- **서비스**: ABA 수퍼비전 (6개월 / 12회기 / 월 2회)
- **가격**: 1:1 수퍼비전 10만원/회, 그룹 5만원/회 (1기 베타)
- **대상**: ABA 대학원생, 초보 치료사
- **수퍼바이저**: 강훈
- **연락처**: myulcan@gmail.com
- **현재 단계**: 문제파악 - 인터뷰 진행 중

## 관련 문서 (Notion)
- 수퍼비전 사업 계획 플랜 (6개월 커리큘럼 상세)
- 프로젝트 허브 (전체 프로젝트 관리)

## 금지 사항
- index.html의 기존 디자인 톤앤매너를 임의로 변경하지 않기
- 한국어 콘텐츠를 영어로 바꾸지 않기
- 외부 JS 라이브러리를 무분별하게 추가하지 않기 (가벼운 정적 사이트 유지)
