# Calendly 설정 가이드 — 인터뷰 예약

## 1. 가입 및 연동 (2분)

1. https://calendly.com 접속 → Google 계정으로 가입
2. 가입 후 설정에서:
   - **Google 캘린더 연동** (자동 빈 시간 표시)
   - **Zoom 연동** (예약 시 Zoom 링크 자동 생성)

## 2. 이벤트 생성 (3분)

### 기본 설정
- **이벤트 이름**: ABA 수퍼비전 경험 인터뷰
- **시간**: 40분
- **장소**: Zoom (연동 후 자동 선택)
- **설명**:
  ```
  안녕하세요! ABA 수퍼비전 경험에 대한 짧은 인터뷰입니다.
  편하게 이야기 나눠주시면 됩니다. 감사합니다!
  ```

### 가능 시간 설정
- 본인이 인터뷰 가능한 요일/시간대만 열기
- 예: 평일 19:00~21:00, 주말 10:00~17:00

## 3. 사전 질문 추가 (Invitee Questions)

이벤트 설정 → "Invitee Questions" 에서 추가:

### Q1. 어디에 해당하시나요? (드롭다운, 필수)
- 센터/개별 치료 외 ABA 관련 활동 + 수퍼비전 (자격증 준비 중)
- 치료실 치료사 + 수퍼비전 (자격증 준비 중)
- 수퍼비전 후 자격증 취득 완료

### Q2. 준비 중이거나 취득하신 자격증은? (한 줄 텍스트, 필수)
- 설명: 예) QBA, BCBA, BCaBA, RBT 등

### Q3. 수퍼비전 형태와 기간 (한 줄 텍스트, 필수)
- 설명: 예) 1:1 수퍼비전, 1년 6개월

### Q4. 연락 가능한 카카오톡 ID 또는 전화번호 (한 줄 텍스트, 필수)

## 4. 완료 후

1. 이벤트 저장
2. 공유 링크 복사 (예: calendly.com/myulcan/aba-interview)
3. 모집글(interview-recruit-draft.md)의 링크 부분에 붙여넣기
4. 블로그에 게시!

## 신청자 경험 요약

```
모집글 → 링크 클릭 → 사전 질문 4개 답변 → 시간 선택 → 확인
→ 양쪽 모두 자동으로: Google 캘린더 등록 + Zoom 링크 + 확인 메일
```
