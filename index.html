What's Different

기존 수퍼비전과 무엇이 다른가요?

훈스아카데미는 실습 환경이 없는 분들을 위해 설계되었습니다.

기존 수퍼비전

센터에 소속돼야 실습 가능

센터를 구하지 못하면 실습 경험을 쌓을 방법이 없습니다.

수퍼바이저 구하기 어려움

소개를 받아도 일정이 안 맞거나, 지역적으로 접근이 어렵습니다.

문제풀기 위주, 체계 없음

자격시험 문제만 풀거나, 매 회기 내용이 뒤죽박죽인 경우가 많습니다.

실제 케이스 없으면 할 게 없음

센터에 다니지 않으면 논의할 케이스 자체가 없어 시간만 흘러갑니다.

훈스아카데미

가상 시나리오로 어디서든 실습

정교하게 설계된 가상 페르소나로 관찰, 측정, 중재를 직접 연습합니다.

1:1 온라인 수퍼비전

Zoom으로 진행하여 어디서든 참여 가능. 개인 맞춤 피드백을 받습니다.

6개월 단계별 커리큘럼

기초(관찰·측정) → 중급(평가·중재) → 심화(종합 케이스)까지 체계적으로.

가상 케이스로 전체 과정 경험

평가 → 중재 → 데이터 분석 → 수정까지, 실제 센터와 동일한 과정을 경험합니다.
